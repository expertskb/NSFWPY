## How to Run Examples

Make sure your virtual environment is active:

```bash
# Run any example with python
.venv/bin/python examples/01_basic_classification.py
.venv/bin/python examples/02_url_classification.py
.venv/bin/python examples/03_batch_classification.py
.venv/bin/python examples/04_pil_and_bytes_classification.py
.venv/bin/python examples/05_custom_model_and_threading.py
.venv/bin/python examples/06_fastapi_server_integration.py
```
